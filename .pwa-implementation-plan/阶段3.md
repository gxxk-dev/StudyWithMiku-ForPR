# 阶段3：测试验证

## 1. 开发环境测试

```bash
npm run dev
```

打开 http://localhost:3000，按 F12 打开开发者工具：

**检查点：**
- Application → Manifest：查看应用清单，图标是否正确
- Application → Service Workers：确认 SW 已注册
- Application → Cache Storage：确认有缓存数据

## 2. 生产构建测试

```bash
npm run build
npm run preview
```

打开 http://localhost:4173

## 3. 离线功能测试

1. 打开 DevTools → Network
2. 勾选 "Offline"
3. 刷新页面
4. 验证：
   - [ ] 页面正常加载
   - [ ] 视频可以播放
   - [ ] 音乐播放器工作
   - [ ] 番茄钟正常计时

## 4. 安装测试

**Chrome/Edge：**
- 地址栏右侧应该出现"安装"图标
- 点击安装，应用以独立窗口打开

**Android：**
- 浏览器菜单 → 添加到主屏幕
- 从主屏幕打开应用

**iOS：**
- Safari → 分享 → 添加到主屏幕

## 5. Lighthouse 审计

在 DevTools：
1. 打开 Lighthouse 标签
2. 勾选 "Progressive Web App"
3. 点击 "Analyze page load"
4. 目标：PWA 评分 100/100
